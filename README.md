# Frontend of the DyHealthNetLight Platform

# Conda environment

Create a conda environment using the requirements.txt file

```bash
conda create -n dyhealthnetlight_frontend
conda activate dyhealthnetlight_frontend
pip install -r requirements.txt
```

Activate your conda environment

```bash
conda activate dyhealthnetlight_frontend
```

# Run the frontend

Be sure that you have the environment file with the correct ports, and the backend is running.

```bash
npm run dev
```
